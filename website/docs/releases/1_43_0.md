# 1.43.0 - 2026-01-22

### Added

* **dbt: Extract test severity from dbt tests** [`#4258`](https://github.com/OpenLineage/OpenLineage/pull/4258) [@mobuchowski](https://github.com/mobuchowski)
  *Add support for extracting and reporting severity information from dbt tests in OpenLineage events.*
* **Spec: Add severity field to DataQualityAssertionsDatasetFacet** [`#4257`](https://github.com/OpenLineage/OpenLineage/pull/4257) [@mobuchowski](https://github.com/mobuchowski)
  *Extend the DataQualityAssertionsDatasetFacet with a severity field to indicate the importance level of data quality assertions.*
* **Spark: Add support for DataSourceRDD with Iceberg** [`#4263`](https://github.com/OpenLineage/OpenLineage/pull/4263) [@kchledowski](https://github.com/kchledowski)
  *Enable lineage extraction from DataSourceRDD when reading Iceberg data sources, supporting mixed RDD and DataFrame operations common in AWS Glue environments.*

### Fixed

* **dbt: Fix send-events command argument parsing** [`#4262`](https://github.com/OpenLineage/OpenLineage/pull/4262) [@mobuchowski](https://github.com/mobuchowski)
  *Fix IndexError when running 'dbt-ol send-events' command without additional arguments by properly checking args length.*
* **Spark: Fix BigQuery column-level lineage namespace** [`#4264`](https://github.com/OpenLineage/OpenLineage/pull/4264) [@mobuchowski](https://github.com/mobuchowski)
  *Fix incorrect "namespace" value in BigQuery column-level lineage to match the format used by regular BigQuery dataset collection.*
* **Spark: Fix isDeltaPlan() to handle multiple Spark extensions** [`#4268`](https://github.com/OpenLineage/OpenLineage/pull/4268) [@mobuchowski](https://github.com/mobuchowski)
  *Fix Delta detection failing when multiple Spark extensions are configured (comma-separated), enabling proper event filtering in environments like Azure Fabric or when using Gluten.*
* **Spark: Fix META-INF/services files of relocated packages** [`#4243`](https://github.com/OpenLineage/OpenLineage/pull/4243) [@kchledowski](https://github.com/kchledowski)
  *Fix service provider interface conflicts caused by package relocation not updating META-INF/services configuration files.*
